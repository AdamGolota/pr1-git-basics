# pr1-git-basics

A simple JavaScript project demonstrating basic Git workflow.

## How to run

Make sure you have [Node.js](https://nodejs.org) installed, then:

```bash
node src/main.js
```

## Features

- Hello World output
- `sum(a, b)` — adds two numbers
- `celsiusToFahrenheit(c)` / `fahrenheitToCelsius(f)` — temperature converters
- `calculator(a, op, b)` — basic calculator supporting `+`, `-`, `*`, `/`
