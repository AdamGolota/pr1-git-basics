# pr1-git-basics

A simple JavaScript project demonstrating basic Git workflow.

## How to run

Make sure you have [Node.js](https://nodejs.org) installed, then:

```bash
node src/main.js
```

## About

This project was created as part of a university assignment on Git basics.
It demonstrates initializing a repository, making meaningful commits, branching, merging, and pushing to a remote.

## Commands I used

```bash
git init
git clone <url>
git add .
git commit -m "message"
git checkout -b feature/readme
git merge feature/readme
git push -u origin main
```

## Features

- Hello World output
- `sum(a, b)` — adds two numbers
- `celsiusToFahrenheit(c)` / `fahrenheitToCelsius(f)` — temperature converters
- `calculator(a, op, b)` — basic calculator supporting `+`, `-`, `*`, `/`
