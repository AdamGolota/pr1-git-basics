Про цей проєкт

Цей сайт створено в рамках практичної роботи №3 з дисципліни «Створення Web-сайтів (Front-end)». Мета — засвоїти базову структуру HTML5-документа, семантичні елементи, форми та медіа-контент.

Веб-розробка — одна з найзатребуваніших сфер у сучасній IT-індустрії. Навіть базове розуміння HTML відкриває двері до створення власних веб-проєктів, блогів, портфоліо та додатків. Починаючи з простого текстового документа, поступово ми додаємо стилі (CSS) та поведінку (JavaScript).

Цей проєкт складається з трьох сторінок: головна (поточна), сторінка «Про мене» з медіа-контентом та сторінка «Контакти» з формою зворотного зв'язку.

Що я вивчаю

На першому курсі я активно опановую такі технології та інструменти:

  • HTML5 — структура та семантика веб-сторінок
  • CSS3 — стилізація, flexbox, grid, анімації
  • JavaScript (ES6+) — динамічна поведінка інтерфейсу
  • Git та GitHub — контроль версій та командна робота
  • VS Code — редактор з розширеннями для веб-розробки

Кроки до першого сайту

Щоб створити повноцінну HTML-сторінку з нуля, я дотримувався такого плану:

  1. Визначити структуру проєкту та створити папки
  2. Написати базовий каркас HTML5 з DOCTYPE та мета-тегами
  3. Додати семантичні блоки: header, nav, main, footer
  4. Наповнити сторінку контентом: текст, зображення, таблиця
  5. Створити додаткові сторінки та зв'язати їх навігацією
